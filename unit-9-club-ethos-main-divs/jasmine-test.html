Centering content with CSS is a core required skill if you want to become a web developer.

How can we centre a block element horizontally? Setting the value auto for left and right margin will centre any block level element.

In order to centre inline-block elements, we need to add the text-align property to the parent of the inline-block.

Vertical centring is a bit trickier. There is a CSS property that can help us achieve our task. The transform property with a translateY value will move the element on the Y-axis (Vertical axis) on the screen.
